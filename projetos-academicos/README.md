# Projetos Acadêmicos

Esta pasta reúne projetos, trabalhos e atividades desenvolvidos durante a faculdade.

## Primeiro Semestre em Engenharia de Software

Durante o primeiro semestre do curso de Engenharia de Software, tive contato com disciplinas importantes para a minha formação acadêmica e profissional. Essas matérias ajudaram a construir uma base inicial sobre tecnologia, programação, organização de dados e desenvolvimento de sistemas.

### Fundamentos de Engenharia

Na disciplina de Fundamentos de Engenharia, aprendi conceitos básicos sobre a atuação do engenheiro, resolução de problemas, pensamento crítico e desenvolvimento de projetos. Essa matéria foi importante para entender melhor o papel da engenharia na criação de soluções para a sociedade.

### Banco de Dados

Em Banco de Dados, estudei a importância da organização e armazenamento de informações. Aprendi conceitos como tabelas, registros, chaves, relacionamentos e modelagem de dados. Esse conhecimento é essencial para desenvolver sistemas que utilizam informações de forma segura e organizada.

### Introdução à Computação

Na disciplina de Introdução à Computação, conheci conceitos fundamentais da área de tecnologia, como hardware, software, sistemas operacionais, redes, segurança e funcionamento básico dos computadores. Essa matéria ajudou a entender melhor a base da computação.

### Lógica de Programação

Em Lógica de Programação, desenvolvi o raciocínio lógico necessário para criar algoritmos e resolver problemas usando programação. Aprendi sobre variáveis, estruturas condicionais, laços de repetição, vetores e matrizes. Essa disciplina é uma das bases mais importantes para quem está começando na área de desenvolvimento de software.

### Engenharia de Software

Na disciplina de Engenharia de Software, aprendi sobre o processo de criação de sistemas, desde o levantamento de requisitos até o planejamento, desenvolvimento, testes e manutenção. Essa matéria mostrou a importância de organizar bem um projeto antes de começar a programar.

### Conclusão da Seção

Essas disciplinas contribuíram para minha evolução no primeiro semestre, ajudando na construção de uma base sólida para os próximos períodos do curso. O PortfolioHUB reúne essa trajetória inicial e apresenta os projetos, aprendizados e habilidades desenvolvidas ao longo da minha formação.
